# react-native-progressive-blur

A cross-platform library for real-time progressive blur effects on Android and iOS. It captures the content behind a view, applies GPU-accelerated blur, and supports mask-driven progressive blur, dynamic intensity, and custom shader effects.A cross-platform library for real-time glass and progressive blur effects on Android and iOS. It captures the content behind a view, applies GPU-accelerated blur, and supports mask-driven progressive blur, dynamic intensity, and custom shader effects.

## Installation


```sh
npm install react-native-progressive-blur
```


## Usage


```js
import { multiply } from 'react-native-progressive-blur';

// ...

const result = multiply(3, 7);
```
